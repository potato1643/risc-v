# Overview

Создание тестового стенда для исследования и оценки существующих тестовых наборов для микропроцессоров с архитектурой RISC-V.

### Списки инструментов

- Эмулятор QEMU (for RISC-V) —— Планируется использовать
- Симулятор Spike (the RISC-V ISA Simulator) —— Уже используется
- Симулятор MicroTESK (for RISC-V) —— Уже используется
- Модель PicoRV32 —— Планируется использовать
- Gcov / lcov / genhtml —— Уже используется

### Списки тестовых наборов

- RISC-V Architectural Certification Tests (ACTs)
  
- Imperas RISC-V Test Suite —— Планируется использовать

- RISCV-DV test suite —— Планируется использовать
