## **Паспорт проекта**

## **Аннотация**

RISC-V — популярная архитектура микропроцессоров, ключевыми особенностями которой являются открытость и расширяемость. В настоящее время идет активное развитие и использование RISC-V архитектуры такими крупными компаниями как NVIDIA, Alibaba T-Head, SiFive. Для тестирования создаваемых RISC-V микропроцессоров разрабатываются различные тестовые наборы. В проекте предлагается провести обзор и сравнить существующие тестовые наборы.

## **Цель**

Создание тестового стенда для исследования и оценки существующих тестовых наборов для микропроцессоров с архитектурой RISC-V.

## **Ожидаемые результаты**

- Обзор публикаций по теме исследования.
- Обзор существующих тестовых наборов и генераторов для их создания для архитектуры RISC-V.
- Обзор инструментов и моделей, позволяющих исполнять тестовые программы, полученные на ассемблере.
- Тестовый стенд, включающий в себя инструменты, программные модели, скрипты для запуска тестов и получения статистики по заданным критериям оценки.
- Тезисы и статьи:
    - Поданные и принятые тезисы на «Межвузовская научно-техническая конференция студентов, аспирантов и молодых специалистов им. Е.В. Арменского».
    - Статья и выступление на конференции SYRCoSE (http://syrcose.ispras.ru/)


## **Форма представления результатов**

- Список рассмотренных работ и короткий реферат по ним.
- Список тестовых наборов и генераторов, их описание, скрипты для генерации тестовых программ.
- Тестовый стенд.
- Демонстрация работы.
- **Тезисы и статья, содержащие обзор и исследование тестовых наборов для RISC-V микропроцессоров.**

## Описание

Для достижения цели проекта исследователю предлагается решить следующие задачи:

- Провести обзор и изучить публикации по теме исследования.
- Провести обзор и изучить имеющиеся тестовые наборы и генераторы для их

создания для архитектуры RISC-V.

- Провести обзор и сформировать списки инструментов (например, симулятор

Spike) и моделей микропроцессорных ядер для выполнения тестовых наборов.

- Сформировать критерии оценки тестовых наборов. Изучить возможности

выбранных программных моделей и инструментов для оценки тестовых

наборов.

- Создать тестовый стенд на основе существующего списка инструментов и

моделей.

- На основе полученных данных сравнить тестовые наборы и сформировать

выводы и рекомендации по их использованию.

Для создания тестового стенда предлагается использовать следующие инструменты:

- Эмулятор QEMU (for RISC-V);
- Симулятор Spike (the RISC-V ISA Simulator);
- Симулятор MicroTESK (for RISC-V);
- Модель PicoRV32;
- Другие открытые инструменты и модели.
Для формирования итогового списка инструментов необходимо провести обзор.


Использовать стенда для сравнения следующих тестовых наборов и инструментов для их генерации:

- RISC-V Architecture Test SIG;
- Генератор MicroTESK for RISC-V;
- Imperas RISC-V test suite;
- Другие открытые тестовые наборы и генераторы.
Для формирования итогового списка тестовых наборов необходимо провести обзор.


Тестовый стенд используется для оценки и сравнения тестовых наборов по следующим критериям:

- Поддержка существующих расширение;
- Покрытие спецификаций;
- Покрытие кода моделей;
- Другие варианты.
