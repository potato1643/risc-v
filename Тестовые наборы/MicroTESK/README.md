# MicroTESK: тестирование и сбор покрытия

## Описание

Для формирования тестового набора MicroTESK и сбора покрытия используются следующие группы тестов:

- **examples/**
  Включены подкаталоги:
  `branches`, `labels`, `preparators`, `registers` (кроме `register_reservation_manual` — временно отключён),
  `selfcheck`, `sequences`, `testdata`.
- **algorithms/**
  Все шаблоны работают корректно:
  `integer`, `sorting`.
- **synthetics/**
  Проблемные шаблоны временно отключены:
  `rv32v`, `rvxxx`.

Остальные шаблоны включены для генерации и тестирования.

## Файлы покрытия

Результаты генерации тестов и файлы покрытия сохраняются в папке:

```
coverage_html/
```

## Запуск тестов

Для запуска всех тестов используйте скрипт:

```
./run_microtesk_tests.sh
```

## Очистка результатов

Для удаления всех сгенерированных файлов и отчётов покрытия используйте:

```
make clean
```

или вручную очистите:

```
rm -r -f $MICROTESK_HOME/output/*
rm -r -f coverage_html/*
```

## Инструкции по использованию Makefile

`make all` — тесты по всем трём директориям.

`make run` — запускает `run_microtesk_tests.sh` для полного тестирования и генерации покрытия.

`make clean` — очищает output и coverage_html.
